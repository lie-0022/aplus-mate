# A+ Mate 기능 명세서 (교수님 검토용)

> **A+ Mate는 한 수업 안에서 학생들이 스스로 팀원을 찾아 팀플·스터디·멘토링을 꾸리고, 교수님은 그 과정을 한눈에 지켜보며 공지·설문·과제 수합을 간편하게 하는 모바일 웹 서비스입니다.**
>
> 교수님은 **아무 세팅도 하실 필요가 없습니다.** 수업·수강생 등록은 운영 측에서 미리 다 해 드리고, 교수님은 로그인하면 바로 "내 수업이 어떻게 돌아가는지" 보는 것부터 시작합니다.

---

## 👩‍🎓 학생이 할 수 있는 것

### 1. 가입과 내 프로필 만들기
- 구글 계정으로 1초 로그인
- 학교·학과·학년, **할 줄 아는 것(스킬 태그)** 등록 → 팀원을 고를 때 서로의 강점이 보임
- 내 프로필이 남들에게 어떻게 보이는지 확인

### 2. 같은 수업 팀원 찾기 (핵심)
- 내 수업에 등록한 다른 학생들을 보고 **"커넥트"** 요청 → 상대가 수락하면 팀 결성
- **세 가지 모임**을 같은 수업에서 따로 운영
  - **팀플** (최대 6명) · **스터디** (최대 5명) · **멘토–멘티** (멘토 1 + 멘티 4)
- 나와 **공통 스킬이 많은 학생이 위에 정렬**되고, 겹치는 강점이 강조돼 누구에게 연락할지 판단이 쉬움
- 받은/보낸 요청을 한 곳에서 관리(취소도 가능)

### 3. 팀 활동 관리
- 팀원 명단·연락처(카카오 오픈채팅)를 한 화면에서 (외부 채팅 안전 안내 포함)
- **팀 일정**: 과제 마감·회의를 등록하고 **담당자 지정**, 완료 체크, D-day 표시, **진척도 막대**("우리 얼마나 했지?")
- **팀 메모 보드**: 회의 결정사항·역할 분담을 앱 안에 기록(카톡처럼 흘러가지 않음)
- **AI 보고서 초안**: 주제를 넣으면 우리 팀이 실제로 한 활동을 반영한 보고서 초안을 자동 생성

### 4. 과제 제출과 성장
- 교수님이 만든 **제출 항목**(예: 1차 기획안)에 결과물 링크를 올림
- 팀플이 끝나면 **동료를 익명 평가**(약속·아이디어·마감) → 좋은 평가를 받으면 **신뢰 배지** 획득
- 배지는 공개 프로필에 표시되어, 다음 팀을 고를 때 "믿을 만한 사람"이라는 신호가 됨

### 5. 소통·정보
- 수업 **게시판**(익명 글·댓글, 부적절한 글 신고·삭제)
- 교수님 **공지** 확인, **설문 참여**
- **알림센터**: 내 요청이 수락되거나 새 공지가 오면 알림
- 처음 온 학생을 위한 **시작 가이드**(프로필→수업등록→첫 커넥트 3단계)

## 👨‍🏫 교수님이 할 수 있는 것

### 1. 수업 현황 한눈에 보기 (로그인하면 첫 화면)
- 수강생 수 / 실제로 시작한 학생 수
- 진행 중·완료된 팀 수, **팀에 합류한 학생 vs 아직 팀 못 구한 학생**
- **팀 미배정 학생 명단** — 누가 소외됐는지 콕 집어 보고 독려
- **설문 응답률**, **마감 임박·미제출 과제** 한눈에

### 2. 수강생·팀 파악
- 수강생 **실명·학과·학년·강점**(스킬) 명단
- 팀별 **구성원·진행상태·평가 진행**을 한눈에

### 3. 공지
- 공지 작성 → 수강생 전원에게 자동 알림, 오타 나면 삭제

### 4. 설문 — 직접 만들고 결과 분석
- **5점 척도 · 객관식 · 주관식** 문항을 자유롭게 구성 (또는 **템플릿 한 번에 불러오기**)
- 결과를 **평균·분포 그래프·선택지 집계·주관식 답변**으로 자동 분석
- **CSV로 내려받아** 보고서·학과 제출용으로 활용

### 5. 과제(산출물) 수합 — "걷는 수고" 없이
- **제출 항목(마일스톤)**을 만들면(예: 중간 발표자료) 각 팀이 링크로 제출
- **어느 팀이 냈는지/안 냈는지 표로 한눈에**, 제출물 바로 열어보고 **"확인" 체크**

## 💡 이 서비스로 할 수 있는 일 (활용 예시)

- **팀 구성을 학생 자율에 맡기되, 소외되는 학생은 교수가 파악·개입** → "조 짜기" 부담 없이 공정성 확보
- **중간 산출물을 한 화면에서 수합·확인** → 메일·USB·출력물 없이 진척 관리
- **공지·설문으로 수업 피드백을 즉시 수집·분석** → 강의 개선에 반영
- **동료평가 데이터로 팀 기여도 파악** → 무임승차 식별 보조

## 🙋 교수님께 여쭙고 싶은 것 (피드백 포인트)

1. 교수님 수업에서 **팀플/조별과제**를 운영하실 때 가장 번거로운 점은 무엇인가요? (조 편성? 진척 확인? 평가?)
2. 위 기능 중 **실제로 쓰실 것 같은 것**과 **별로일 것 같은 것**은?
3. **동료평가 결과를 교수님이 직접 열람**(누가 무임승차인지 익명 집계)하는 기능이 있으면 도움이 될까요?
4. 학생 **시간표를 넣어 "공강이 맞는 팀원" 추천**을 더하면 유용할까요?
5. 이 외에 "이런 게 있으면 수업에 쓰겠다" 하는 기능이 있으신가요?
